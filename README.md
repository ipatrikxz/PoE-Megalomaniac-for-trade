## About The Project

The purpose of this project is to provide a summary of all the notable jewel mod names and descriptions on one page, making it easy for you to search through them.

## Usage

- Type keywords into the search bar to search through the names and descriptions.
- Click on a row to add it to your list.
- Delete items from your list by simply clicking on the mod at the top of the page.
- Select a search type.
- Click "Search on trade" - This will generate a search query based on your selected mods and open it in the official trade site in a separate window.

## Features

- Quick searchable list of mod names and their descriptions.
- Selectable mods to create your own list.
- Search types similar to the official trade site (AND, COUNT).
- Generate search queries for the official PoE trade site based on your list and search type.
